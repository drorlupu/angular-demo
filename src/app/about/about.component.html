This is a sample angular app designed to demo some basic angular features.


The resources for this app were taken from:

  • Dog images: Downloaded from Kaggle, which in turn is based on Stanford's ImageNetDogs
  • Cat images: Generated by the awesome This cat doesn't exist!
  • Heart breaking background stories were generated by OpenAI's ChatGPT


Note: All animal information displayed in this web application is fictional.

No animal was hurt/abandoned/mistreated. This is just a demo app!


The angular application is composed of the following features/modules/libraries, which are listed in no order of importance:

  • Inputs/Outputs/Angulr Directives
  • Router
  • Angular Material
  • RxJS
  • Express
On top of that, this represents the knowledge I acquired from the following resources:
  • Angular Deep Dive and RxJS courses
  • Mini SideNav, with cool code


Instructions: Choose up to three lovley animals in the "Friends" tab, then click on the "Meet" tab to set a "match-making" meeting.
